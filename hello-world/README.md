# Bonjour le monde

Bienvenue dans « Bonjour le monde » sur le parcours Go d'Exercism.
Si vous avez besoin d'aide pour exécuter les tests ou soumettre votre code, consultez `HELP.md`.

## Instructions

L'exercice d'introduction classique.
Dites simplement « Bonjour, tout le monde ! ».

["Hello, World!"][hello-world] est le premier programme traditionnel permettant de commencer à programmer dans un nouveau langage ou un nouveau environnement.

Les objectifs sont simples :

- Modifiez le code fourni pour qu'il produise la chaîne "Hello, World!".
- Exécutez la suite de tests et assurez-vous qu'elle réussit.
- Soumettez votre solution et vérifiez-la sur le site Web.

Si tout se passe bien, vous serez prêt à aller chercher votre premier véritable exercice.

[hello-world]: https://en.wikipedia.org/wiki/%22Hello,_world!%22_program
